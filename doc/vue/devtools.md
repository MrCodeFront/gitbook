# Devtools 安装及使用

------

##### 克隆官方项目

```bash
git clone https://github.com/vuejs/devtools.git
```

##### 安装依赖

```bash
# 官方使用的是 yarn，所以使用 yarn 进行安装，否则会报错
yarn install
```

##### 打包

```
yarn run build
```

##### 使用

> chrome 更多工具 -> 扩展程序 -> 启用开发者模式 -> 加载已解压的扩展程序，把项目的 packages/shell-chrome 文件夹导入即可。
