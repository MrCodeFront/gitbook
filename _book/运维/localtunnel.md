## localtunnel

github地址：https://github.com/localtunnel/localtunnel



localtunnel将您的本地主机公开给世界，以方便测试和共享！无需为了让其他人测试您的更改而破坏DNS或进行部署。

相当于创建临时公网隧道



##### 安装

```bash
npm install -g localtunnel
```

```bash
yarn add localtunnel
```



##### 使用

```bash
npx localtunnel --port 8000
```

全局安装可用

```bash
lt --port 8000
```
